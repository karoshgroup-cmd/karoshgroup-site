# Karosh Group Site Pack (Netlify + GitHub)

## What you get
- `index.html` : the full design (HTML + CSS in one file)
- `content/home.json` : all main text + card info (editable)
- `admin/` : Netlify CMS setup so you can edit `home.json` from `/admin`
- `_redirects` : makes Netlify routes work

## Photos / Images
Put your real images in `assets/` with EXACT names:
- `hero-bg.jpg`
- `card-studio.jpg`
- `card-wedding.jpg`
- `card-agency.jpg`
- `card-event.jpg`

## Upload steps
1) GitHub repo → **Add file → Upload files**
2) Upload EVERYTHING from this pack (folders + files)
3) Commit
4) Netlify deploy → refresh your site

## Edit text later (CMS)
Go to `/admin/` on your domain and login (Netlify Identity must be enabled).
